The factory fixing itself

Scars — regressions it introduced, then closed

Autonomy means owning your own mistakes. Each of these was a self-inflicted break the factory diagnosed and repaired — feedstock for the failure-to-eval flywheel.

Jun 5#224
OR-join trigger_rule self-skip

Branch-merge nodes silently skipped when a sibling skipped; forced none_failed_min_one_success + a structural lint guard.

Jun 11#250
E2BIG bash-node inlining

Archon inlined node output into a bash argv; fixed by reading node outputs from artifact files.

Jun 12#397
classify-preview silent work-discard

An unparseable when: fail-closed-skipped, letting implement exit 0 and discard work; made it fail loudly instead.

Jun 12#362
compose duplicate-key regression

Two factory PRs each adding the same compose key produced an unparseable duplicate volumes: on main.

Jun 13#106
Re-merged a settled migration

Factory re-introduced schema drift a migration had already fixed; restored the orphaned regime migration.

Jun 14#436
BuildKit 403 through socket-proxy

preview-up died on every implement; resolved with remote BuildKit + migrate service.

Jun 15#389
Scheduler strands off-board deps

dependencies_met() stranded tickets whose dependency was closed but off-board.

Jun 19#560
False main-red latch

Smoke-gate import check missing REDIS_PASSWORD falsely latched main-red and paused all dispatch.

Jun 26#536 · #622
Python 3.14 dep break

Base-image bump made scipy source-build with no cp314 wheel, killing every run at pip install.

Jun 26#522
Memory-write data loss

Memory writes could delete valid entries; added an append-only guard.
